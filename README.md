# doda-types 👻

> 개인 TypeScript Definitions 저장소.

## 왜 만듬?

- 외부 라이브러리의 타입 선언
- [DefinitelyTyped](https://github.com/DefinitelyTyped/DefinitelyTyped)에 없는게 많음
- 다른 타입 선언의 수정 (사유: 오래됨, 누락됨 등)

## 구조

```
📁 types: TypeScript Definitions
  📁 libraries: 외부 라이브러리
  📁 packages: NPM 등록 패키지
📁 templates: 그냥 템플릿
```
